NITC Lost & Found


This web application is called Found It.
Basic motive of this project is decrease the amount of lost and found mails in nitc email.
People who lost their stuff can post details of the item in lost page.
If anyone found any item they can post about the item they found in found page.
Along with this there will also be a second hand selling page.
